

## Hacker News Clone with react and firebase
React,TypescriptでHackerNewsのクローンを作成しました。

#### 使用技術
* React
* Redux
* ReduxThunk
* Firebase(firestore,Authentication)
* React-router
* react-loading
* HackerNews API (https://github.com/HackerNews/API)




#### `下記のURLに公開しております。`
https://react-hacker-news-90ba0.web.app/



##### `主な機能`

* 各種投稿の表示
* ユーザーのプロフィールを表示
* 検索機能(タイトルのみ)
* ブックマーク機能
